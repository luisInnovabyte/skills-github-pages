# ¡Bienvenido a Mi Sitio Web!

Este es un sitio web estático creado con GitHub Pages como prueba de concepto.

## Acerca de Este Sitio

GitHub Pages permite convertir repositorios de GitHub en sitios web. Este sitio fue creado utilizando Jekyll, un generador de sitios estáticos.

## Características

- **Fácil de usar**: Solo necesitas archivos Markdown
- **Gratuito**: Hospedado directamente en GitHub
- **Personalizable**: Usa temas de Jekyll y tu propio contenido
